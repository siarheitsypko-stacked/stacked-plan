🌐 Языковая среда CSS

CSS (Cascading Style Sheets) — язык оформления веб-страниц, работающий в связке с HTML и JavaScript. Вся мощь CSS реализуется через браузерные движки (Blink, Gecko, WebKit), которые интерпретируют стили и отображают их. Экосистема включает препроцессоры, фреймворки, инструменты оптимизации и современные спецификации W3C.

⚙️ Браузерные движки Blink (Chrome, Edge), Gecko (Firefox), WebKit (Safari) — отвечают за рендеринг CSS.
📐 Препроцессоры Sass, Less, Stylus — добавляют переменные, вложенность, миксины, компилируются в обычный CSS.
🛠️ Инструменты разработчика DevTools в браузерах: инспектирование, вычисленные стили, отладка анимаций, профилирование производительности.
📦 Фреймворки и библиотеки Tailwind CSS (утилитарный), Bootstrap, Bulma, Materialize — ускоряют разработку интерфейсов.
🚀 Пост-процессинг PostCSS с плагинами (Autoprefixer, CSS Modules, PurgeCSS) для кросс-браузерности и оптимизации.
📏 Современные спецификации CSS Grid, Flexbox, Container Queries, Layers, :has() — постоянно эволюционирующий стандарт.

📘 Основы CSS
Синтаксис, подключение, каскадность, единицы измерения (px, rem, em, vw, vh).
/* селектор + правило */
h1 {
  color: #007acc;
  font-size: 2rem;
}
.container {
  width: 80vw;
  padding: 1.5rem;
}
🔍 Селекторы
Базовые, комбинаторы, псевдоклассы / псевдоэлементы.
.card .title { }      /* потомок */
section > p { }        /* прямой потомок */
a:hover { opacity: 0.8; }
li:nth-child(odd) { }
p::before { content: "✨"; }
🎨 Цвета и фон
RGB, HEX, HSL, градиенты, фоновые изображения.
color: #1e3a8a;
background: rgba(0,122,204,0.2);
background: linear-gradient(135deg, #007acc, #1e3a8a);
background: url('bg.png') center/cover no-repeat;
✍️ Текст & шрифты
Шрифты, выравнивание, трансформация, веб-шрифты.
body {
  font-family: 'Inter', sans-serif;
  font-size: 1rem;
  font-weight: 500;
  line-height: 1.5;
  text-align: center;
  text-transform: capitalize;
}
📦 Box Model
content, padding, border, margin, box-sizing.
.box {
  width: 300px;
  padding: 20px;
  border: 2px solid #007acc;
  margin: 15px;
  box-sizing: border-box; /* ширина включает padding+border */
}
📍 Позиционирование
static, relative, absolute, fixed, sticky, z-index.
.parent { position: relative; }
.child {
  position: absolute;
  top: 0;
  right: 0;
  z-index: 10;
}
.header { position: sticky; top: 0; }
🧩 Flexbox
Гибкая раскладка, выравнивание, управление осями.
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}
.item { flex: 1 1 200px; order: 2; }
🔲 Grid Layout
Двумерная сетка, fr, grid-template-areas.
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
  grid-template-areas:
    "header header header"
    "sidebar main main";
}
.item { grid-column: span 2; }
📱 Адаптивность
Медиа-запросы, Mobile First, viewport.
/* mobile-first */
.container { padding: 1rem; }
@media (min-width: 768px) {
  .container { max-width: 720px; margin: auto; }
}
@media (min-width: 1024px) {
  .grid { grid-template-columns: repeat(4, 1fr); }
}
✨ Анимации & переходы
transition, keyframes, transform.
.btn {
  transition: all 0.3s ease;
}
.btn:hover {
  transform: scale(1.05);
  background: #007acc;
}
@keyframes slide {
  from { opacity: 0; transform: translateX(-20px);}
  to { opacity: 1; transform: translateX(0);}
}
.fade-in { animation: slide 0.5s forwards; }
🎛️ CSS переменные
Custom Properties, наследование, динамические темы.
:root {
  --primary: #007acc;
  --spacing: 1rem;
}
.card {
  background: var(--primary, #0a58ca);
  padding: var(--spacing);
}
.dark { --primary: #1e2a3a; }
🚀 Современный CSS
clamp(), :has(), container queries, subgrid.
/* адаптивный размер */
h1 { font-size: clamp(1.5rem, 5vw, 3rem); }
/* родительский селектор */
article:has(img) { padding: 1rem; }
/* контейнерные запросы */
@container (min-width: 400px) {
  .card { grid-template-columns: 1fr 1fr; }
}
🛠️ Экосистема
Препроцессоры, PostCSS, CSS-in-JS, фреймворки.
/* Sass (SCSS) */
$primary: #007acc;
.button { background: $primary; }

/* Tailwind Utility */
<div class="bg-blue-600 p-4 rounded">

/* Styled Components */
const Button = styled.div`
  background: #007acc;
`;
