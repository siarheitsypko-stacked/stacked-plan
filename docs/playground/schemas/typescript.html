📐 TypeScript Mind Map

Статическая типизация, интерфейсы, дженерики, утилитарные типы и мощные инструменты
🔷 Базовые типы

Примитивы: boolean, number, string, null, undefined, symbol, bigint. Массивы (number[] или Array<number>), кортежи ([string, number]), enum, any, unknown, void.

let isDone: boolean = false;
let age: number = 42;
let name: string = "TypeScript";
let list: number[] = [1, 2, 3];
let tuple: [string, number] = ["hello", 10];
enum Color { Red, Green, Blue }
let notSure: unknown = 4;
let nothing: void = undefined;

⚙️ Продвинутые типы

Union & Intersection, Literal Types, Type Guards, Mapped Types.

type ID = string | number;
let userId: ID = "abc123";

type Point = { x: number; y: number } & { color: string };
const p: Point = { x: 10, y: 20, color: "red" };

type Direction = "up" | "down";
function move(dir: Direction) { }

function isString(value: unknown): value is string {
    return typeof value === "string";
}

type ReadonlyPoint = { readonly [P in keyof Point]: Point[P] };

📄 Интерфейсы и типы

interface – описание формы объекта, расширяется (extends). type alias – может описывать примитивы, union, intersection.

interface User {
    id: number;
    name: string;
    email?: string;   // опциональное поле
    readonly createdAt: Date;
}

type Point = { x: number; y: number };
type ID = string | number;

interface Admin extends User {
    role: "admin";
}

🏛️ Классы и ООП

Модификаторы доступа: public, private, protected, readonly. Абстрактные классы, сокращённый конструктор, реализация интерфейсов.

abstract class Animal {
    constructor(public name: string) {}
    abstract speak(): void;
}

class Dog extends Animal {
    speak() { console.log(`${this.name} гавкает`); }
}

interface ILogger {
    log(msg: string): void;
}
class ConsoleLogger implements ILogger {
    log(msg: string) { console.log(msg); }
}

🧬 Дженерики (Generics)

Обобщённые функции, классы, интерфейсы. Ограничения (extends).

function identity<T>(arg: T): T {
    return arg;
}
let output = identity("myString");

interface GenericBox<T> {
    value: T;
}
class Box<T> {
    constructor(public content: T) {}
}
function logLength<T extends { length: number }>(item: T): T {
    console.log(item.length);
    return item;
}

🛠️ Утилитарные типы

Partial<T>, Required<T>, Readonly<T>, Pick<T, K>, Omit<T, K>, Record<K, T>, Exclude<T, U>, ReturnType<T>.

interface Todo {
    title: string;
    description: string;
    completed: boolean;
}
type PartialTodo = Partial;
type ReadonlyTodo = Readonly;
type TodoPreview = Pick;
type TodoWithoutDesc = Omit;
type Users = Record;

function getValue() { return 42; }
type Return = ReturnType; // number

📦 Модули и конфигурация

ES Modules, tsconfig.json – ключевые опции (strict, target, module, outDir, rootDir).

// math.ts
export const add = (a: number, b: number) => a + b;

// app.ts
import { add } from "./math.js";
console.log(add(5, 3));

// tsconfig.json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "include": ["src/**/*"]
}

const button = document.querySelector("button")!;
button.addEventListener("click", (e: MouseEvent) => {

🚀 Продвинутые приёмы

Conditional Types, Template Literal Types, декораторы (экспериментальные), типизация this.

type IsString = T extends string ? true : false;
type A = IsString<"hello">; // true
type B = IsString;  // false

type EventName = `on${Capitalize}`;
// EventName = "onClick" | "onChange" | ...

function fancyThis(this: { name: string }) {
    console.log(this.name);
}
