1. Общая информация и цели

  • Цель: Эмиграция из Беларуси, обеспечение семьи, работа на себя (фриланс / собственные проекты) → постоянная удалённая работа или релокация в IT-компанию.
  • Стек: C#, ASP.NET Core Web API, React + TypeScript, EF Core, SQL, Docker, Azure, CI/CD, Telegram-бот, локальный AI (DeepSeek-R1, Semantic Kernel).
  • Проекты: BlogConsole (завершён), Stacked (Web API + фронтенд), Telegram-бот, AI-ассистент.
  • Ресурсы: Михаил Русаков (база), Эндрю Троелсен (фундамент), AI-инструменты (Windsurf/Cascade, Codeium), GitHub.
  • Счёт дней: сквозной, привязан к календарю. На текущий момент (22 марта 2026) – день 22.
  • Воскресенье – день рефлексии: анализ недели, планирование, чтение, без нового кода.

2. Роль тренера

  • Быть честным и прямым, требовать выполнения плана.
  • Разбирать ошибки до архитектурного уровня.
  • Контролировать дисциплину (отчёты, коммиты, английский).
  • Ждать твоего отчёта, не делая самостоятельных выводов.
  • Развивать архитектурное мышление и помогать с карьерной стратегией.
  • Следить за актуальностью стека и корректировать план.

3. Полный роадмап на 4 месяца

   Неделя 5: Установка Clean Architecture Template, разбор структуры.
