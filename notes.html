📝 Stacked – Заметки и полезные ресурсы

Ссылки, глоссарий, технический английский и вспомогательные материалы для обучения

📚 Основные ресурсы обучения

  • Михаил Русаков – «C# с нуля до гуру» (основной курс, практика).
  • Эндрю Троелсен – «C# 7 и .NET Core» (8-е издание, чтение в метро 30 мин/день).
  • Сергей Дмитриевский – «Профессия Frontend-разработчик + ИИ» (используем выборочно с TypeScript).
  • Metanit – быстрый справочник по C# и .NET.
  • learn.microsoft.com – официальная документация Microsoft.
  • CodeBlog (YouTube) – русскоязычный канал по C#.
  • Сачков Dev (YouTube) – полезные уроки по C# и .NET.
  • Codewars – задачи для тренировки алгоритмического мышления (с 3-4 недели).

📖 Технический английский – базовый глоссарий

string – строка
int – целое число
double – число с плавающей точкой
bool – логический тип
if / else – условие
switch / case – множественный выбор
while / for – циклы
array – массив
List<T> – список
class / object – класс / объект
method – метод
property – свойство
interface – интерфейс
repository – репозиторий
dependency injection – внедрение зависимостей
commit / push / pull – операции Git

*Глоссарий пополняется по мере изучения новых тем.

🛠️ Инструменты разработки

  • IDE: Visual Studio Community 2026 (основная), VS Code (для фронтенда)
  • Git: версия 2.53.0, GitHub Desktop для удобства
  • .NET SDK: 8.0 (LTS) и 10.0 (для совместимости)
  • Docker Desktop – для контейнеризации (будет использоваться на 4-м месяце)
  • Ollama + DeepSeek-R1 – локальный AI-ассистент (для проекта и обучения)

⏰ Ежедневный ритуал (напоминалка)

  • 📖 30 минут Троелсена в метро.
  • 🎯 Утром: определить цель дня (из детального плана или роадмапа).
  • 💻 Основная работа: теория (Русаков) + практика (BlogConsole / API).
  • 🌙 Вечером: отчёт «день X итог» (что сделано, трудности, долги).
  • 📝 Коммиты и пуш на GitHub – ежедневно.
  • 📚 Английские термины – учить по 5-10 слов в день.
