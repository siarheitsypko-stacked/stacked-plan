📟 BlogConsole активный

C#.NET 8JSONRepository patternLINQGit

Консольное приложение для управления постами блога. Полный CRUD, сохранение в JSON, чистая архитектура (интерфейсы, репозиторий). Фундамент для всех будущих проектов.

GitHub

🔧 Составляющие роадмапа:

  • Основы C# (переменные, циклы, условия) – неделя 1
  • ООП: классы, методы, интерфейсы, статика – неделя 2
  • Коллекции (List, Dictionary), LINQ – неделя 2–3
  • Работа с файлами (JSON, Stream) – неделя 3
  • Исключения, обработка ошибок – неделя 3
  • Первый репозиторий, коммиты, Git – день 1–2

⚙️ Stacked API в процессе

ASP.NET CoreClean ArchitectureCQRSEF CorePostgreSQLSwagger

Веб-API для блог-платформы. Слои: Domain, Application, Infrastructure, Web. Используется CQRS (MediatR), FluentValidation. В будущем — клиент на React и интеграция с GitHub API.

Репозиторий (план) В профиле

🔧 Составляющие роадмапа:

  • Entity Framework Core, миграции, SQLite/PostgreSQL – месяц 1, неделя 3
  • Web API: контроллеры, маршрутизация, DI – месяц 1, неделя 3–4
  • JWT аутентификация, роли – месяц 2, неделя 8
  • GitHub API интеграция – месяц 3, неделя 9
  • React + TypeScript фронтенд – месяц 2, недели 5–8
  • Docker, CI/CD – месяц 4, неделя 13
  • Azure, деплой – месяц 4, неделя 14
  • Тестирование (xUnit, интеграционные тесты) – месяц 4, неделя 15

🤖 Telegram‑бот (AI assistant) в разработке

C#Telegram.BotAI (Semantic Kernel)OllamaDeepSeek-R1Docker

Ассистент для работы с блогом через Telegram. Создание/просмотр постов, интеграция с локальной AI‑моделью. Демонстрирует навыки интеграции с внешними API и гибридных AI‑сценариев.

🔗 Как всё связано

  • BlogConsole – фундамент C# и первое знакомство с кодом.
  • Stacked API – профессиональный бэкенд, интеграции, архитектура.
  • Telegram‑бот + AI – демонстрация AI‑сценариев и работы с внешними интерфейсами.
  • Все дополнительные этапы роадмапа (GitHub API, React, Docker, Azure, фриланс-модули) либо уже встроены в эти три проекта, либо являются их естественным продолжением.
