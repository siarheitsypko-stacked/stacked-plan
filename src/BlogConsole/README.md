# BlogConsole — консольный менеджер блога на C#

[![.NET Version](https://img.shields.io/badge/.NET-8.0-blueviolet)](https://dotnet.microsoft.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**BlogConsole** — это учебный проект, демонстрирующий основы объектно-ориентированного программирования на C#, работу с коллекциями, сериализацию JSON, а также применение паттерна Repository и базовых принципов чистой архитектуры. Проект представляет собой консольное приложение для управления постами блога.

## 🚀 Возможности

- ✅ Создание нового поста с валидацией (заголовок не менее 3 символов, содержание не пустое)
- ✅ Просмотр всех постов с уникальными ID, заголовками, содержимым и датой создания
- ✅ Редактирование существующих постов
- ✅ Удаление постов с подтверждением
- ✅ Автоматическое сохранение постов в JSON-файл (`posts.json`) и загрузка при старте
- ✅ Чистая архитектура: разделение на модели, репозиторий и пользовательский интерфейс
- ✅ Интерфейс `IPostRepository` для лёгкой замены хранилища (например, на базу данных)

## 🛠️ Технологии

- **C#** (.NET 8)
- **.NET SDK** 8.0
- **System.Text.Json** для сериализации
- **LINQ** для работы с коллекциями
- **Git** и **GitHub** для управления версиями
